# Colors

Save colors, create new one and combine them, so they fit perfectly. 

This project was created as a practice of html, css and js. As well I was frustrated to save fancy colors as hex codes.

## Functions 

  - enter color
  - save, delete color 
  - mix colors
  - copy color
